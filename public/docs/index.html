Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://127.0.0.1:8000

Authenticating requests

This API is authenticated by sending an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Endpoints

POST api/login

requires authentication

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/login" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"phone\": 29334758.1981,
    \"password\": \"doloremque\"
}"

const url = new URL(
    "http://127.0.0.1:8000/api/login"
);

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "phone": 29334758.1981,
    "password": "doloremque"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

POST api/register

requires authentication

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/register" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "email=mcdermott.rosetta@example.com" \
    --form "name=eos" \
    --form "phone=5685922.3203" \
    --form "password=dolorem" \
    --form "bio=atque" \
    --form "avatar=@/tmp/phpBz2ZJ5" 
const url = new URL(
    "http://127.0.0.1:8000/api/register"
);

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('email', 'mcdermott.rosetta@example.com');
body.append('name', 'eos');
body.append('phone', '5685922.3203');
body.append('password', 'dolorem');
body.append('bio', 'atque');
body.append('avatar', document.querySelector('input[name="avatar"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
